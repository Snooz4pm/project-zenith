# Job Mapping: Paths → Careers → Skills → Modules

This document defines the actionable "Career Calibration" layer. It connects abstract Paths to concrete job roles and the specific learning modules required to achieve them.

## 1️⃣ Market Analyst
**Focus:** Logic, Structure, Reasoning.

### Critical Roles
1.  **Equity Research Associate:** Analyzes company fundamentals to recommend buy/sell.
2.  **Macro Strategy Analyst:** Predicts broad market moves based on economic data.
3.  **Investment Committee Support:** Prepares data-driven memos for fund decision-makers.

| Skill | Learning Module | Quiz ID |

| **Fundamental Analysis** | *Valuation 101: DCF & Multiples* | `valuation-basics` |
| **Financial Modeling** | *Excel for Finance Pros* | `financial-modeling` |
| **Macroeconomic Theory** | *Central Banks & Interest Rates* | `macro-economics` |
| **Thesis Writing** | *Structuring the Investment Memo* | `investment-writing` |
